關於我

我是 廖宥驄，一位專注於介面與程式架構的前端工程師——相信好的畫面，來自好的結構。

大三因專題合作進入哈瑪星科技實習，第一次接觸到課堂外的 Angular 與 Git 版本控制，從此踏上前端這條路。實習期間奠定基礎後，持續自學 RxJs、NgRx 等進階技術，把它們實際用在企業級系統上。

我喜歡把複雜、零散的需求，拆解成模組化、可重用的元件；也習慣在開發過程中重構與優化既有程式碼，讓系統更好讀、更好維護。從 Angular 生態一路延伸到 React 與 Next.js，我關注的始終是同一件事：用清晰的架構，做出讓人信任的介面。
